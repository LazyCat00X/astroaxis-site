Lord’s Cricket Ground set for first-ever women’s Test as England host India

  • 英格蘭將於週五在倫敦的洛德板球場舉行首場女子測試賽，對手為印度隊，這是該場地歷史上首次舉辦女子測試賽。
  • 此次比賽距離洛德板球場首次舉辦男子測試賽已過去142年，並且是自1976年以來首次舉辦女子比賽。
  • 英格蘭隊的隊長Nat Sciver-Brunt因小腿受傷仍希望參加比賽，九名女子世界盃成員將參加此次測試賽。
  • 英格蘭教練Charlotte Edwards表示這場比賽對於英國和印度隊來說都是歷史性時刻，期待在接下來的四天內吸引大量觀眾。
